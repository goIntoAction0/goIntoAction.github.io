## 一、git常用命令


### 1.1 最常用的三个git命令
``` sh
// 将代码提价到暂存区
git add .

// 提交暂存区到仓库区
git commit -m [message]

// 将代码提交到远程仓库
git push
```




### 1.1新建git代码库
``` sh
# 在当前目录新建一个 Git 代码库
$ git init

# 新建一个目录，将其初始化为 Git 代码库
$ git init [project-name]

# 下载一个项目和它的整个代码历史
$ git clone [url]
```
