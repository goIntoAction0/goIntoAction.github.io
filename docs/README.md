## 为什么要写博客：
#### 原因一：避免重复学习
 1. 很多东西如果不记录下来，很容易就忘了。 
 2. 避免重复，为了提高学习效率。因为，我发现，我常学了忘，做了过多且没有必要的重复劳动。

#### 原因二：编程工具的安装
1. 经常需要安装一些编程工具，安装的次数少，但是如果忘记流程，就会浪费很多事件。
2. 如果有笔记，就可以避免不必要的浪费。
